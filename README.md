Predictive Factors of MTCT of HIV/AIDS — Logistic Regression Model
Overview
This project develops a logistic regression model to identify statistically significant predictors of Mother-to-Child Transmission (MTCT) of HIV/AIDS using Kenya Demographic and Health Survey (KDHS) data.
Problem Statement
Despite significant progress in Prevention of Mother-to-Child Transmission (PMTCT) programmes across sub-Saharan Africa, MTCT of HIV remains a critical public health challenge in Kenya. Identifying the strongest predictive factors enables more targeted intervention and resource allocation.
Methodology

Data source: Kenya National Health Survey (KDHS)
Tools: Python (Pandas, NumPy, scikit-learn, matplotlib, seaborn)
Approach: Data cleaning → EDA → Feature engineering → Logistic Regression → Model evaluation

Role: Team Lead — responsible for project scoping, data pipeline, modelling, and findings presentation
Key outputs: Identification of statistically significant determinants of MTCT with policy implications for PMTCT programme targeting
